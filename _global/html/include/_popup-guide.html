윈도우 팝업

웹 접근성 지침에 따라 사용자에게 새창열림 공지 및 키보드 포커싱 관련 처리가 필수이며, 다음과 같은 방식으로 처리합니다.

윈도우 팝업 호출

  • 앵커태그(a) 사용을 원칙으로 하며 버튼에 target="_blank" title="새창" 어트리뷰트 필수
  • href에 팝업 URI를 입력하고, openWindow(this, '팝업창이름', '팝업너비', '팝업높이'); return false; 함수 사용
링크로 윈도우팝업 호출 (권장) <a href="_popup.html" target="_blank" title="새창" onclick="openWindow(this, '팝업창이름', '팝업너비', '팝업높이'); return false;">팝업열기</a> openWindow('popup.html', 'popName', '850', '600', '#btn-for-focus-popup');

버튼없이 팝업을 호출하는 경우에만 사용하며, 부득이 버튼에 사용할 경우 포커싱처리 및 title="새창" 필수

openWindow('팝업URI', '팝업창이름', '팝업너비', '팝업높이', '팝업창 닫은 뒤 포커스 받을 버튼 id (버튼 미존재시 생략가능)');

윈도우 팝업 닫기

팝업을 닫은 뒤 호출 버튼으로 포커싱 처리를 위한 함수

<button type="button" onclick="closeWindow();">창닫기</button>
